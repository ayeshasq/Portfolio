## üì¶ Deployment to Vercel

### Method 1: GitHub Repository (Recommended)

1. **Create a new GitHub repository**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/portfolio.git
   git push -u origin main
   ```

2. **Deploy on Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign in with your GitHub account
   - Click "New Project"
   - Import your portfolio repository
   - Click "Deploy"

### Method 2: Direct Upload

1. **Install Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **Deploy**
   ```bash
   vercel
   ```

3. **Follow the prompts** and your site will be live!

## üé® Customization

### Update Your Information

Open `index.html` and update:

1. **Meta Tags** (Line 5-7):
   ```html
   <title>Your Name | Cybersecurity Engineer</title>
   ```

2. **Hero Section** (Line 150+):
   - Name
   - Subtitle
   - Tagline

3. **About Section** (Line 170+):
   - Your bio
   - Statistics

4. **Projects** (Line 300+):
   - Add/remove projects
   - Update GitHub links
   - Modify descriptions

5. **Contact Information** (Line 500+):
   - Email
   - Phone
   - Social links

### Color Scheme

Modify CSS variables in the `:root` section:

```css
:root {
    --primary: #00ff88;        /* Main accent color */
    --accent: #ff3366;         /* Secondary accent */
    --bg-dark: #0a0e1a;       /* Background */
    --text-primary: #e8eaed;  /* Main text */
}
```
